# ChronoSkipper

Submission for the [Unity 20th Anniversity Game Jam](https://itch.io/jam/unity-20th-anniversary-game-jam) made in 48 hours.

Submission link: https://p4gan1.itch.io/chronoskipper

*While pondering your orb, you seem to accidentally have broken time in a most peculiar way. Time has stopped. But only while you're on the ground. Will you be able to fix it?*

Controls:
- WASD/Arrow keys to move
- N or C to jump
- M or X to float
